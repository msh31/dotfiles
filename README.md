# Marco's MacOS dotfiles
This repository contains all my configurations for the tools that I use on my Macbook

## Included
- Ghostty theme
- Tmux remappings (and Sessionizer from theprimeagen)
- Neovim remaps & plugins
- zsh w/ aliases, helper funcs & QOL stuff 
- Aerospace config
- Brew package list (often outdated)
